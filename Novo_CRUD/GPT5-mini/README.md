# Students CRUD (Python)

Simple Python script that implements a CRUD for students with JSON persistence.

Files:
- [students_crud.py](students_crud.py): library + interactive CLI.
- [test_run.py](test_run.py): non-interactive demo that registers, updates and deletes students.

Usage:

Run the interactive CLI:

```bash
python students_crud.py
```

Run the demo:

```bash
python test_run.py
```

Data is stored in `students.json` in the same folder.
