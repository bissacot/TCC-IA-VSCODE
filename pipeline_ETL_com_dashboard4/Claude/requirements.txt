## Installation

### Production Installation
```bash
pip install -r requirements.txt
```

### Development Installation (with dev dependencies)
```bash
pip install -r requirements-dev.txt
```

### From Source
```bash
# Create virtual environment
python -m venv venv

# Activate virtual environment
# On Windows:
venv\Scripts\activate
# On macOS/Linux:
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt
```

## Troubleshooting

### Common Installation Issues

**Issue**: `psycopg2` installation fails
```bash
# Solution: Use binary version
pip install psycopg2-binary
```

**Issue**: `Pillow` build fails
```bash
# Solution: Install system dependencies
# Ubuntu/Debian
sudo apt-get install libjpeg-dev zlib1g-dev

# macOS
brew install libjpeg zlib
```

**Issue**: `lxml` compilation fails
```bash
# Solution: Install system dependencies
# Ubuntu/Debian
sudo apt-get install libxml2-dev libxslt1-dev

# macOS
brew install libxml2 libxslt
```

## Dependency Security

Check for security vulnerabilities:
```bash
pip install safety

```

Update dependencies:
```bash
pip list --outdated
pip install --upgrade [package-name]
```

## System Requirements

- **Python**: 3.9+ (3.11+ recommended)
- **PostgreSQL**: 12.0+ (14+ recommended)
- **RAM**: Minimum 2GB, 4GB+ recommended
- **Disk**: At least 1GB free space
- **Network**: For API connectivity
