💻 EXEMPLO DE USO RÁPIDO (Python)
═══════════════════════════════════════════════════════════════════════════════

from Compact_Encoding_System import LinhaProducao

# Criar linha
linha = LinhaProducao("Linha Premium")

# Criar Ordem de Fabricação
of = linha.criar_ordem_fabricacao(ano=25, tipo_linha=1, codigo_cliente=12)
print(of.gerar_codigo())  # 25112

# Criar Ordem de Produção
op = linha.criar_ordem_producao(
    codigo_of="25112",
    fase=1,
    subfase=2,
    modo_recurso="B",
    recurso_base=5
)

# Simular execução
simulacao = op.simular_execucao()
print(simulacao['recursos_utilizados'])  # [5, 6]

🎯 VALIDAÇÕES IMPLEMENTADAS
═══════════════════════════════════════════════════════════════════════════════

✓ Formato OF (5 caracteres YYTCC)
✓ Formato OP (10 caracteres YYTCCFSMRB)
✓ Ano: 00-99 (reduz com mod 100)
✓ Tipo: 0 ou 1 (Normal/Premium)
✓ Cliente: 00-99
✓ Fase: 0-2
✓ Subfase: 0-2
✓ Modo: A, B ou C
✓ Recurso Base: 00-26
✓ Acumulação Correta:
  • A (1 recurso):     [base]
  • B (2 recursos):    [base, base+1]
  • C (3 recursos):    [base, base+1, base+2]
✓ Proteção Overflow: (base+num_recursos-1) ≤ 26
